# From Canvas to Code Apps: Your Next Step (It's Easier Than You Think)

**East of England Power Platform Summit (EoEPPS)**
**1 May 2026 | Hosted by Charles Sexton & Josh Giles**

Welcome! If you're here, you already know how to build great things with Canvas Apps. Today we'll take that expertise and channel it into something new — your first **Power Platform Code App**, built with a little help from **GitHub Copilot**.

No prior coding experience needed. Seriously. You'll be surprised how much you already know.

---

## Before You Arrive

Please complete these steps **before the workshop day** so we can hit the ground running. If you get stuck on any of them, don't worry — we'll have time at the start to troubleshoot.

### Accounts You'll Need

- [ ] **Microsoft 365 account** with access to a Power Platform environment (your org account is fine)
- [ ] **GitHub account** — [sign up free](https://github.com/signup) if you don't have one
- [ ] **GitHub Copilot access** — your organisation may already provide this, or you can [start a free trial](https://github.com/features/copilot)

### Software to Install

- [ ] **Visual Studio Code** — [download here](https://code.visualstudio.com/)
- [ ] **GitHub Copilot extension for VS Code** — install from the Extensions panel inside VS Code
- [ ] **Power Platform CLI (pac)** — we'll walk through this in [the setup guide](docs/SETUP.md)
- [ ] **Node.js (LTS version)** — [download here](https://nodejs.org/)
- [ ] **Git** — [download here](https://git-scm.com/downloads)

> **First time installing any of these?** Our [step-by-step setup guide](docs/SETUP.md) walks you through everything, one click at a time.
