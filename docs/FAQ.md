# Frequently Asked Questions

## Before the Workshop

**Q: Do I need to know how to code?**
A: Nope! If you can build a Canvas App, you have all the thinking skills you need. We'll teach you the rest.

**Q: What should I bring?**
A: Your laptop (fully charged), your Microsoft 365 credentials, and a willingness to experiment. A mouse is handy too — trackpads work fine, but a mouse can be more comfortable for a full day.

**Q: I couldn't finish the setup guide — should I still come?**
A: Absolutely. We've built in time at the start of the day to help everyone get set up. Just get as far as you can and bring your laptop.

**Q: Will I need internet access?**
A: Yes — we'll be connecting to Power Platform environments and using GitHub Copilot, both of which need an internet connection. Wi-Fi details will be provided on the day.

---

## During the Workshop

**Q: I'm falling behind — what should I do?**
A: Raise your hand! We have helpers in the room and we'd much rather pause and help you than have you struggle in silence. There's no prize for finishing first.

**Q: Can I use my own Dataverse environment?**
A: We'll provide a shared dev environment, but if you have your own and prefer to use it, that's absolutely fine. Just make sure you have maker permissions.

**Q: GitHub Copilot is suggesting something I don't understand — should I accept it?**
A: Treat Copilot like a helpful colleague — it's often right, but always read what it suggests before accepting. If you're unsure, ask us. That's exactly the kind of thing we want to discuss.

**Q: What if I break something?**
A: That's what dev environments are for! You genuinely cannot break anything important today. Experiment freely.

---

## After the Workshop

**Q: Can I keep using the code we built?**
A: Yes — everything in this repo is yours. Fork it, clone it, share it with your team.

**Q: Where can I learn more about Code Apps?**
A: Check out our [links and resources](../resources/links.md) page for official docs, community links, and learning paths.

**Q: I have a question that isn't answered here — where do I ask?**
A: [Open an issue](../../issues/new/choose) on this repo and we'll get back to you.

**Q: Will there be a recording?**
A: The workshop is hands-on, so there won't be a traditional recording. However, all the materials, guides, and code remain available in this repo indefinitely.
